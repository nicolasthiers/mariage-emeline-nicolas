# Site de mariage - Émeline & Nicolas

Ce dépôt contient le site web de notre mariage, développé avec React, TypeScript, Vite et Tailwind CSS.

Le site permet de présenter les informations importantes liées à l’événement et de centraliser les réponses des invités via un formulaire RSVP connecté à une base de données Supabase.

## Fonctionnalités

- Présentation du mariage
- Informations pratiques pour les invités
- Formulaire de réponse RSVP
- Enregistrement des réponses dans Supabase
- Interface responsive adaptée mobile et desktop
- Déploiement en ligne via Vercel

## Technologies utilisées

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Supabase
- Vercel
